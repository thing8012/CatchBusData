# Catch Bus data and insert to mongodb

### Step 1
Clone project

```bash
git clone https://github.com/thing8012/CatchBusData.git
```    
### Step 2
```bash
pip install -r requirements
```
### Step 3
setting app_id and app_key secret in config.yaml
setting mongodb host and port in config.yaml

### Step 4
start
```bash
python getData.py
```
